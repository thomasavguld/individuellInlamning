Version 3: Med stöd för att databasfrågor til SQLite och visning av Markdown


Använd inte LiveServer för att visa webbsidan: 

öppna terminalen i VSCode
	npm start + ENTER för att starta webbservern.
    	Ctrl + C för att stänga

Nytt:

    En funktion du kan inkludera för att ställa databasfrågor (endast SELECT-queries): dbQuery. 
    (Notera: Du behöver ställa in vilken databas som ska användas i filen sqlite-databases/database-in-use.json och databasen måste finnas i mappen sqlite-databases.)

    En funktion du kan inkludera för att visa upp markdown som HTML i webbläsaren: addMdToPage.
    
    Förbättring av funktionen tableFromData, så att egna kolumnnamn ej behöver anges, om du inte vill.
